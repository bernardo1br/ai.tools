Uso de ferramentas de IA e risco de treinamento com dados

Use este painel para decidir quando é aceitável inserir dados em ferramentas de IA. A regra prática é simples: contas individuais exigem cautela máxima; planos corporativos com acordo específico sobre tratamento de dados pessoais e proibição de treinamento são preferenciais para uso profissional.

0preferenciais

Não usar

Dados pessoais sensíveis, documentos de clientes, segredos comerciais, dados estratégicos ou informações sujeitas a sigilo profissional em contas individuais.

Usar com controles

Dados internos de baixo risco somente com anonimização, minimização, opt-out de treinamento e configuração adequada de histórico/retenção.

Preferir

Planos corporativos com acordo específico sobre tratamento de dados pessoais, proibição contratual de treinamento, retenção definida, suboperadores mapeados e controles de acesso corporativos.

Matriz de decisão rápida

Revise os termos oficiais e o contrato vigente antes de inserir dados pessoais, confidenciais ou de clientes.
